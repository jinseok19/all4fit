신청 방법

간단한 절차로 바우처를 신청하고 이용하세요

1

대상자 확인

각 바우처별 지원 대상에 해당하는지 확인합니다.

• 소득 기준 확인
• 연령 조건 확인
• 거주지 확인
2

신청서 작성

온라인 또는 방문 신청을 통해 서류를 제출합니다.

• 신청서 작성
• 필요 서류 준비
• 온라인/방문 제출
3

심사 및 승인

제출한 서류를 바탕으로 심사 후 승인 여부를 통지합니다.

• 서류 심사 (7~14일)
• 승인 통지
• 바우처 발급
4

이용 시작

발급받은 바우처로 원하는 체육 프로그램을 이용합니다.

• 시설/프로그램 선택
• 바우처 번호 입력
• 이용 시작

자주 묻는 질문

국민체육진흥공단과 협약을 맺은 전국의 체육시설에서 사용 가능합니다. 모두의핏 모든 시설에서 바우처 사용이 가능하며, 홈페이지에서 '바우처 사용 가능' 표시가 된 프로그램을 이용하실 수 있습니다.

스포츠강좌이용권 공식 홈페이지(svoucher.kspo.or.kr)에서 로그인 후 확인하거나, 모바일 앱 '스포츠강좌이용권'을 통해 실시간으로 잔액을 확인할 수 있습니다.

바우처는 지정된 사용 기간 내에만 사용 가능하며, 기간이 지나면 자동으로 소멸됩니다. 미사용 잔액에 대한 환불은 불가하므로 사용 기간을 반드시 확인하시고 계획적으로 이용해 주세요.

바우처는 본인만 사용 가능하며, 가족 간 양도나 공동 사용은 불가합니다. 다만, 가족 구성원 각각이 신청 조건에 해당한다면 개별적으로 신청하여 사용할 수 있습니다.

바우처는 주로 그룹 강좌에 사용 가능하며, 개인 레슨의 경우 시설별로 정책이 다를 수 있습니다. 이용하고자 하는 시설에 미리 문의하여 개인 레슨 바우처 사용 가능 여부를 확인해 주세요.
